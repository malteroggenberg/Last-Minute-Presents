# Last Minute Presents 🎁

This repository contains small, fast-to-build 3D printed gifts designed for situations where time is limited — birthdays, Christmas, or other occasions where you still want to create something personal.

## Idea
I often end up building presents on short notice.  
Instead of buying something generic, I design and print simple but meaningful objects that can be produced quickly.

The focus is not on perfect optimization, but on:
- fast design
- reliable printing
- practical results within limited time

## What you'll find here
- Quick 3D printable gift ideas
- Designs that can be finished within hours (not days)
- Simple concepts that can be customized easily

## Philosophy
Most designs are created under time pressure, so they are not always fully optimized.

Over time, I aim to:
- improve printability
- reduce print time
- refine designs
- document what works (and what doesn’t)

## Examples
- Christmas lamp (designed & printed within ~24h)
- Small personalized objects
- Functional decorative parts

## Goal
To build a collection of simple, reproducible and practical gift ideas  
→ useful when you need something **fast, but still personal**

## Note
This is an evolving collection.  
Some designs may be rough, but they are tested in real situations.
